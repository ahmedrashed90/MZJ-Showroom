# MZJ Showroom Screens v25

نسخة مبنية على v23 لأن v24 تم إلغاؤها.

## الجديد
- سحب الألوان الفعلية للسيارة من بيانات الموقع وصفحة السيارة قدر الإمكان.
- قائمة اللون في الداش بورد لم تعد ثابتة؛ تظهر حسب ألوان السيارة نفسها عند الضغط على "الصور/اللون" ثم "تحميل الصور".
- اختيار الصور اليدوي اختياري فقط. لو لم تحدد صور، الشاشة تستخدم اللون المختار.
- لو فلتر اللون لم يجد صورًا مطابقة في أسماء الصور، الشاشة تعرض الصور المتاحة بدل ما تبقى فاضية.
- الحفاظ على تشغيل Vercel: `/dashboard` و `/screen?id=A1`.

## الاستخدام
ارفع محتويات هذا الفولدر على GitHub ثم اعمل Redeploy على Vercel.
