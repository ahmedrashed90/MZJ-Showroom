## صفحة PDF الجديدة

تمت إضافة صفحة تصدير PDF:

- `/pdf?id=A1`
- `/export?id=A1`

الصفحة تعرض صورة السيارة والمواصفات الرئيسية، وبها زر `تصدير PDF`.
يمكن فتحها من زر PDF في معاينة الداش بورد أو مباشرة من الرابط.
