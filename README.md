# MZJ Showroom Screens v26

نسخة Vercel جاهزة للنشر.

## الجديد في v26

- إصلاح اختيار اللون: النظام يحاول الآن قراءة Color Variations الفعلية من صفحة السيارة في الموقع.
- يدعم ألوان الموقع بصيغة hex مثل `#dbdbdb` و `#ffffff` بدل الاعتماد على أسماء ثابتة فقط.
- لو صفحة السيارة فيها صور مرتبطة بكل لون، الشاشة تعرض صور اللون المختار مباشرة.
- قائمة الألوان في الداش بورد تظهر حسب السيارة نفسها، ومعها عدد صور اللون عند توفره.
- عند الضغط على تحميل الصور في الداش بورد يتم تحديث بيانات السيارة من الموقع مباشرة، وليس من كاش Firestore القديم.
- تم إلغاء الاعتماد على v24 والبدء من v25 مع إصلاح الألوان الفعلية.

## الروابط بعد النشر

- الإدارة: `/dashboard`
- الشاشة: `/screen?id=A1`

## طريقة التحديث

ارفع محتويات فولدر هذه النسخة إلى GitHub ثم اعمل Redeploy على Vercel.

بعد النشر افتح الداش بورد، اختر السيارة، اضغط `الصور/اللون` ثم `تحميل الصور` لتظهر ألوان السيارة الفعلية من الموقع.
